Add support for auto embedding VoiceThreads.

== Description ==

This plugin will allow you to embed a VoiceThread just by pasting the link into your post or page.

== Installation ==

1. Upload the `voicethread-auto-embed` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Enter your settings in the VoiceThread Auto Embed options page.
1. Paste a link to a VoiceThread (e. g., http://voicethread.com/?#q.b409) into a post or page
